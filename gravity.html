Planetary Motion

Code --> python 
from turtle import *
import math
dot(70,"red")
up()
setposition(300,0)
down()
color("blue","blue")
shape("circle")
loop = True
xvel = 0
yvel = 1
while loop == True:
    xvel +=  math.cos(math.radians(towards(0,0)))*(1000/(xcor()**2+ycor()**2))
    yvel +=  math.sin(math.radians(towards(0,0)))*(1000/(xcor()**2+ycor()**2))
    setposition(xcor()+xvel,ycor()+yvel)

This Python code uses the Turtle graphics library to simulate the motion of a blue circle being influenced by the gravitational attraction of a red dot at the center (0, 0) of the window.

Here's a step-by-step explanation of the code:

Import necessary modules:

The code imports the `turtle` module, and all the functions are directly imported into the namespace, so you can use them without the `turtle.` prefix.

Draw the red dot:

The `dot()` function is used to draw a red dot with a diameter of 70 pixels at the center (0, 0) of the window.

Set initial position and shape for the blue circle:

The turtle's pen is lifted (`up()`), and the turtle is moved to the position (300, 0) without drawing (`setposition(300, 0)`). The pen is put down (`down()`), and the turtle's shape is set to a blue circle (`shape("circle")`).

Initialize variables:

The variables `loop`, `xvel`, and `yvel` are initialized. The variable `loop` is set to `True`, which will allow the animation loop to run indefinitely. The `xvel` and `yvel` variables represent the initial x and y components of the velocity of the blue circle, respectively.

Animation loop:

The code enters an infinite loop (`while loop == True`) to continuously update the position of the blue circle based on the gravitational attraction of the red dot.

Update velocity components:

In each iteration of the loop, the `xvel` and `yvel` variables are updated to simulate the gravitational attraction. The angle towards the red dot is calculated using the `towards(0, 0)` function, which returns the angle (in degrees) from the current turtle position to the point (0, 0) (the position of the red dot). The `math.cos()` and `math.sin()` functions are used to calculate the x and y components of the velocity, respectively. The velocity is inversely proportional to the distance from the red dot, with a scaling factor of 1000.

Update position:

The turtle's position is updated by adding the x and y components of the velocity to the current x and y coordinates using `setposition(xcor() + xvel, ycor() + yvel)`.

The blue circle will start at the position (300, 0) and be affected by the gravitational attraction of the red dot. The circle's path will be influenced by the gravitational pull, resulting in an animated motion as it moves closer to the red dot. The simulation will continue indefinitely due to the infinite loop.
